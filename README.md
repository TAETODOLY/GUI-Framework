1. New Project Class Library named "WebDriverProvider"
	- Classes
	- Enums
	- Interfaces
	- Configurations
	- Extensions
1. New Project Class Library named "ConfigurationProvider"
	- Classes
1. New Project NUnit named "TestSolution"
	- Features
	- StepDefinitions
	- Hooks
	- PageObjects
	- Models
	- Configurations
	- TestData
1. Install Following NuGet Packages
   - "Selenium"
   - "WebDriverManager"
